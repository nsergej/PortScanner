# Fast TCP Port Scanner for Windows

High performance multithreaded TCP port scanner for Windows.

Scan any IP address and detect open ports with accurate response time measurement.  
Designed for system administrators, network engineers and security testing.

---

## Purpose

This tool allows fast and reliable detection of open TCP ports on a target host.  
It provides a simple graphical interface and generates structured reports.

---

## Key Advantages

- Multithreaded scanning  
- Simple graphical interface  
- Accurate detection of open ports  
- Real time progress display  
- No installation required  

---

## Features

- Full port range scanning from 1 to 65535  
- Single IP scanning  
- Multithreaded processing  
- Non blocking sockets using WinSock  
- Response time measurement  
- Automatic sorting of open ports  
- Export reports to HTML, CSV, JSON  
- Clean and responsive interface

## Usage

1. Run PortScanner.exe  
2. Enter target IP address  
3. Set port range if required  
4. Click Start  

Scan progress and results are displayed in real time.

- Audit local network devices  
- Identify exposed services
